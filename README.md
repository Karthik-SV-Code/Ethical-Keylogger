# 🖥️ Ethical Keylogger – Keyboard Activity Monitor

## 📌 Project Overview
This project is an **Ethical Keylogger** developed for educational purposes. It captures and logs keyboard inputs **only within the application window**, ensuring privacy and ethical usage.

The application demonstrates key concepts such as:
- Event-driven programming
- GUI development
- Keyboard input handling
- File logging (JSON format)

---

## 🎯 Features
- Real-time key press detection
- Timestamp-based logging
- Scrollable log display
- Start and Stop logging controls
- Clear log functionality
- JSON file storage
- Safe and ethical implementation (no system-wide tracking)

---

## 🛠️ Technologies Used
- Python
- Tkinter (GUI framework)
- JSON (data storage)
- Event Binding (`<Key>` events)
- File Handling

---

## 🚀 Installation & Setup

### 1. Clone the Repository
```bash
git clone https://github.com/Karthik-SV-Code/ethical-keylogger.git
cd ethical-keylogger
```

---

## ▶️ Run the Application

### Step 1: Navigate to Project Folder
```bash
cd ethical-keylogger
```

### Step 2: Run the Python File
```bash
python main.py
```

### Step 3: Use the Application
- Click **Start** to begin logging  
- Type inside the application window  
- View logs in real time  
- Click **Stop** to stop logging  
- Click **Clear Log** to reset logs  

---

## 📂 Output
- `key_log.json` → Stores structured keystroke data with timestamps

---

## ⚠️ Ethical Considerations
- This project is strictly for **educational purposes**
- Does NOT capture system-wide keystrokes
- Works only within the application window
- Ensures user privacy and consent
